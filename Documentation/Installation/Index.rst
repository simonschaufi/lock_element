Installation
============

The extension needs to be installed as any other extension of TYPO3 CMS:

#. **Use composer**: Use `composer require simonschaufi/lock-element`.

#. **Get it from the Extension Manager:** Press the "Retrieve/Update" button and search for the extension key *lock_element* and import the extension from the repository.

#. **Get it from typo3.org:** You can always get current version from `https://extensions.typo3.org/extension/lock_element/ <https://extensions.typo3.org/extension/lock_element/>`_ by downloading the zip version. Upload the file afterwards in the Extension Manager.
